name: react-ts-component
description: Use this skill when creating or refactoring React TypeScript components in the pizzeria project.
---

When working with React + TypeScript components:

- Use interface for props.
- Prefer simple function components.
- Keep logic readable and minimal.
- Use regular SCSS files.
- Do not use module.scss.
- Do not add styling libraries.
- Do not add new dependencies unless explicitly requested.
- Keep reusable components in the appropriate components directory.
- Explain changes briefly after editing.
